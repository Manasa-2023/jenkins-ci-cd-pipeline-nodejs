# Jenkins CI/CD Pipeline - Node.js App

## Description
This project demonstrates a simple Jenkins CI/CD pipeline for a Node.js app using Docker.

## Features
- Node.js web server using Express
- Jenkins pipeline with:
  - Clone
  - Build (`npm install`)
  - Test (dummy)
  - Docker Build
  - Docker Run

## How to Use
1. Clone the repo
2. Run the app manually:
   ```bash
   npm install
   npm start
